# ShopClone - React E-commerce Application

A modern e-commerce application built with React, TypeScript, and Vite, featuring a clean UI powered by shadcn/ui components and TailwindCSS.

## Features

### Product Browsing
- Category-based product navigation
- Responsive product grid layout
- Product search functionality
- Advanced sorting options:
  - Price (low to high)
  - Price (high to low)
  - Rating

### Product Details
- Modal-based product view
- Product images and descriptions
- Quantity selection
- Add to cart functionality
- Rating display

### Shopping Cart
- Persistent cart state
- Add/remove items
- Quantity management
- Real-time total calculation
- Popup cart view

### UI/UX Features
- Responsive design
- Dark mode support
- Clean, modern interface
- Pagination (12 items per page)

## Tech Stack

- **Framework:** React 18 with TypeScript
- **Build Tool:** Vite
- **Styling:** TailwindCSS
- **UI Components:** shadcn/ui (built on Radix UI)
- **Routing:** React Router DOM
- **State Management:** React Hooks (useState)





## Getting Started

1. Clone the repository
2. Install dependencies:
```bash
npm install
```
3. Start the development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```

## Component Library

This project uses shadcn/ui components, which provide:
- Accessible and customizable components
- Built on top of Radix UI primitives
- Seamless integration with TailwindCSS
- Dark mode support

Key components used:
- Card - Product displays
- Dialog - Product details modal
- Sheet - Mobile navigation
- Popover - Shopping cart
- Input - Search functionality
- Select - Sorting options
- Pagination - Product page navigation

## Configuration Files

The project includes several configuration files:
- `vite.config.ts` - Vite build configuration
- `tailwind.config.js` - TailwindCSS styling configuration
- `tsconfig.json` - TypeScript configuration
- `eslint.config.js` - Code linting rules
